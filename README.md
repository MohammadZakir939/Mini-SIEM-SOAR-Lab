# Mini SIEM + SOAR Lab

## Overview

A Security Operations Center (SOC) automation lab built using:

- Sysmon
- Splunk Enterprise
- Splunk Universal Forwarder
- Python
- MITRE ATT&CK Framework

## Features

- Log Collection
- Detection Engineering
- Alert Correlation
- Risk Scoring
- Incident Creation
- Case Management
- Investigation Reports
- SOAR Automation

Sysmon
